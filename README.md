# apple-remote-panasonic-tv

> Purpose

Use an Apple remote (https://support.apple.com/en-bh/HT200131) on a Panasonic Viera TV.

> Mockup

A mockup is done with an Arduino Uno, an ir sensor and a 940nm ir led.
